VYTDLP – Visual YT-DLP Wrapper
==============================

Versión: 1.0  
Autor: fedegure
Licencia: Open Source (consultar disclaimer.txt para detalles legales)

Descripción
-----------
VYTDLP es una herramienta visual y portable para descargar contenido usando yt-dlp.exe.  
Pensada para usuarios no técnicos, ofrece una interfaz profesional y moderna, sin dependencias externas.

Requisitos
----------
- Windows 7 o superior  
- yt-dlp.exe en la misma carpeta que VYTDLP.exe  
- No requiere instalación ni Python

Cómo usar
---------
1. Colocá `yt-dlp.exe` en el mismo directorio que `VYTDLP.exe`.  
2. Ejecutá `VYTDLP.exe` haciendo doble clic.  
3. Pegá la URL del video/audio que quieras descargar.  
4. Elegí el formato y destino.  
5. Presioná "Descargar" y dejá que la magia ocurra.

Archivos incluidos
------------------
- VYTDLP.exe         → Interfaz gráfica principal  
- yt-dlp.exe         → Motor externo para descarga  
- icon.ico           → Icono personalizado (opcional)  
- disclaimer.txt     → Aclaraciones legales y términos de uso  
- README.txt         → Este documento

Notas importantes
-----------------
- yt-dlp puede actualizarse fácilmente: reemplazá `yt-dlp.exe` sin recompilar VYTDLP.  
- Si yt-dlp no está presente, VYTDLP mostrará un mensaje claro y descriptivo.  
- Antivirus puede marcar el `.exe` como falso positivo: asegurate de que venga desde una fuente confiable.
